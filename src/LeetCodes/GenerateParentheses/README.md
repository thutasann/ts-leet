# GenerateParentheses

## Question

Give `n` paris of parentheses, <br/>
write a function to generate all combinations of well-formed parentheses

> well-fomred means `()` not `)()`

## Examples

- Given n = 3, a solution set is

```javascript
Output: ['((()))', '(()())', '(())()', '()(())', '()()()'];
```

- Given n = 1, a solution set is

```javascript
Output: ['()'];
```
